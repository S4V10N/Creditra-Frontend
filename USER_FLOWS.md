## 8. Repayment Validation Error

```
┌─────────────────────────────────────────────────────────────────┐
│                   Repayment Modal (Step 1)                      │
│                User Enters Amount > Total Due                   │
│             OR Amount > Available Wallet Balance                │
└────────────────────────────┬────────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────────┐
│                     Error Indicators Show                       │
│  - Input border turns Red                                       │
│  - "Amount exceeds total due" or "Insufficient balance"         │
│  - [Review Repayment] button becomes Disabled                   │
└─────────────────────────────────────────────────────────────────┘
```
